# Drago Launcher

A safe, clear, and custom Minecraft Launcher built with Python and CustomTkinter. 

## Features
- **Skins Manager:** Easily apply custom skins and see them in-game.
- **Modrinth Browser:** Browse, search, and install Minecraft mods directly from Modrinth within the launcher.
- **Installed Content Manager:** Easily manage and delete installed mods and worlds.
- **News Feed:** See the latest official Minecraft news.
- **Custom RAM Allocation:** Dynamically fetch PC specs and set max RAM for Minecraft Java.
